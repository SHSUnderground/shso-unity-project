So yesterday I was having great diffculty finding the jean grey FX file. I finally decided to find out if there
was a way of using grep to search for more than one string in a file, and it seems you can. So let me show you 
what I did, and you can use it as a template for finding other FX files.
First, go to Assets\GameData\Characters and open jean_grey.xml in a text editor. Look for a block that starts 
with <character_fx>. There might be several effects listed, choose one of the prefab_names of an effect. In my 
case, I chose Fly Trail. So now we know that the FX cab has to have at least one occurance of jean_grey AND at 
least one occurance of Fly Trail. The grep command (run from the same directory that CABS are located) to do
this search is:     grep -i -l jean_grey CustomAssetBundle* | xargs grep -i -l 'Fly Trail'
So, to find an FX for a different character, the only things you have to change in the grep command are the 
character name (jean_grey) and the fx prefab name (Fly Trail). BTW, this command actually found 6 files that 
matched, so I had to look at each one....it was fairly obvious after looking at all their contents which one
was the FX file.
